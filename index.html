Toggler

A toogler allows you to add a toggling functionality to an element.

  • What will happen:
  • Clicking on the button will always toggle the box class.
  • But clicking on the box (the target) won't trigger the toggle.
  • But clicking outside the box (body) will untoggle the box if it has been toggled.
